有了镜像才可以创建容器，拉取一个最新的centos镜像

    # docker pull contos

启动容器：

    #  docker run [OPTIONS] IMAGE [COMMAND] [ARG...]

    [OPTIONS]常用参数：
    --name="自定义name"     容器名字 tomcat01 tomcat02
    -d                    通过后台的方式运行（Run container in background and print container ID）
    -it                   使用交互的方式，进入容器查看内容
    -p                    指定容器端口 -p 主机端口:容器端口
    -P                    随机指定端口

    # 启动并进入容器
    [root@localhost image]# docker run -it --name 'myCentos' centos /bin/bash

    # 查看正在运行的docker容器 docker ps [OPTIONS]
    [root@localhost ~]# docker ps
    CONTAINER ID   IMAGE     COMMAND       CREATED         STATUS         PORTS     NAMES
    fa342784818c   centos    "/bin/bash"   8 seconds ago   Up 7 seconds             myCentos2
    b745d512744c   centos    "/bin/bash"   2 minutes ago   Up 2 minutes             myCentos

    [OPTIONS]常用参数：
    -a      历史运行过的所有容器
    -n=?    列出最近运行过的？个容器
    -q      只显示容器的编号

    # 退出容器
    # exit
    # 容器不停止并退出
    # ctrl + P + Q

    # 删除容器
    # docker rm 容器id
    # docker rm -f $(docker ps -aq)

    # 启停容器
    docker start 容器id
    docker restart 容器id
    docker stop 容器id
    docker kill 容器id
