1.进入centos根目录执行以下命令（\ 是linux系统种命令换行符，如果命令过长，可以用\来换行）

    yum remove docker \
    docker-client \
    docker-client-latest \
    docker- common \
    docker-latest \
    docker-latest-logrotate \
    docker-logrotate \
    docker-sqlinux \
    docker-engine-selinux \
    docker-engine \
    docker-ce


2.安装需要的安装包

    yum install -y yum-utils \

3.设置镜像仓库

    yum-config-manager \
    --add-repo \
    http://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

4.更新软件包索引

    yum makecache fast

5.安装docker
    docker-ce社区版本 ee企业版
    sudo yum install docker-ce docker-ce-cli containerd.io docker-compose-plugin

6. 启动docker

    systemctl start docker

7. 查看docker

    docker version

    docker run hello-world
